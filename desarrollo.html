Desarrollo Movil
Este proyecto tuvo como objetivo abordar el tema del desarrollo móvil. Para comprender este punto, hemos creado una aplicación móvil con la herramienta FlutterFlow, la cual es de maquetado code (utiliza código), pero que dispone de una interfaz que simplifica su uso. Esta actividad nos permitió ver las diferentes herramientas que podemos ocupar a la hora de desarrollar aplicaciones mobiles. Asimismo, los tipos de desarrollo móvil disponibles.

DESARROLLO MÓVIL

Del desarrollo móvil se derivan tres tipos de aplicaciones: las nativas (lenguajes determinados), las PWA (construidas es HTML5, JS y CSS) y las hibridas (adaptables a cualquier plataforma).  Hoy en día solo existen dos sistemas operativos que utilizan el desarrollo móvil, iOS y Android. Dependiendo para que sistema vayas a desarrollar tendrás disponibles diversas herramientas; las más comunes son: Firebase, Xamarin, Flutter y React. A su vez, hay que tener en cuenta que clase de maquetado usaras, code(tienes que programar), nocode(no tienes que programar) y lowcode(lo que programaras es mínimo.) el tipo de maquetado dependerá de tus necesidades.
