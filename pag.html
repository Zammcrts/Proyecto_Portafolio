Página web
Esta página engloba dos tareas, las cuales radicaron en explorar los conceptos de una página web y un repositorio. Con el fin de comprenderlos, se optó por elaborar una página web acerca de nosotros. Este fue creado utilizando HTML, CSS (Frontend). Posteriormente, introducimos el concepto de un repositorio, para tener nuestro proyecto en la nube y poder acceder a él a libre demanda. Para ello, nos apoyamos de la herramienta Git y de la plataforma GitHub, en la cual alojamos nuestro proyecto y posteriormente nos apoyamos para poder publicarlo.

PAGINA WEB

Para la página web; utilizamos las etiquetas de HTML y para los estilos de cada página CSS (Frontend). A su vez utilizamos bibliotecas públicas, como lo fue Boostrap, la cual nos fue de mucha ayuda a la hora de generar componentes, como por ejemplo la Navbar. También trabajamos de la mano con los generadores de contenedores, para ayudarnos con los divs.

GIT

Utilizamos Git que es un sistema de control de versiones distribuido, lo que significa que un clon local del proyecto es un repositorio de control de versiones completo. Esta herramienta nos ayuda a tener nuestro proyecto a nuestro alcance dondequiera que estemos.


PUBLICACIÓN

GitHub nos permitió crear el repositorio y guardarlos en la nube de forma segura, usando Git. En esta plataforma se encuentra todo el código de nuestro proyecto, y es accesible para todos. De igual forma, gracias a sus servicios, hicimos el deploy de nuestra página, para publicarla.
